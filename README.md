# StreamNexus Updates

This repository hosts StreamNexus release artifacts and the source used to operate the official website.

StreamNexus is a Windows desktop application for Twitch and YouTube stream monitoring, notifications, and multiview playback.

## Official links

- Website: <https://stream-nexus.com/>
- Release notes: <https://stream-nexus.com/releases/>
- Downloads and release notes: <https://github.com/rodojp/StreamNexus-updates/releases>
- Support: <https://stream-nexus.com/support/>
- Contact: <https://stream-nexus.com/contact/>
- Privacy Policy: <https://stream-nexus.com/privacy/>
- Terms of Service: <https://stream-nexus.com/terms/>

## Release channels

- Stable releases are published as regular GitHub Releases.
- Beta releases are published as GitHub prereleases.
- Installers and update metadata are attached to each applicable release.

Always download StreamNexus from this repository's [Releases](https://github.com/rodojp/StreamNexus-updates/releases) page.
