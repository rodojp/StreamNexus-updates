StreamNexus 利用規約

発効日: 2026年6月2日 · 最終更新日: 2026年6月12日

この利用規約は、StreamNexus デスクトップアプリおよびそのリリース配布に適用されます。

1. StreamNexus の利用

StreamNexus は、Twitch と YouTube の配信監視、通知、マルチビュー再生を行う Windows デスクトップアプリです。 ユーザーは、適用される法律および連携する各サービスの規約に従って StreamNexus を利用する責任を負います。

2. Twitch、Google、YouTube 連携

YouTube 連携を利用する場合、以下も適用されます。

  • YouTube 利用規約
  • Google プライバシーポリシー
  • StreamNexus プライバシーポリシー

StreamNexus は読み取り専用 YouTube OAuth スコープ https://www.googleapis.com/auth/youtube.readonlyのみを使用します。 StreamNexus は、YouTube 動画、コメント、プレイリスト、またはチャンネルのアップロード、削除、編集、管理を行いません。

3. 禁止事項

ユーザーは、StreamNexus を以下の目的で使用してはなりません。

  • 連携サービスの規約または API ポリシーに違反すること。
  • 不正アクセスを試みること。
  • API キー、OAuth クライアントシークレット、OAuth トークン、その他の認証情報を共有または公開すること。
  • 外部 API に対して不正または過剰なリクエストを送信すること。
  • 他者のプライバシー、セキュリティ、または権利を侵害すること。

4. 可用性と外部サービス

StreamNexus は、外部 API、OAuth プロバイダー、配信プラットフォーム、オペレーティングシステムの挙動、および Electron に依存します。これらの外部システムが変更された場合、機能が利用できなくなったり、更新が必要になったりする場合があります。

5. 変更

StreamNexus は、アプリ機能、リリース運用、またはコンプライアンス要件が変わった場合、この規約を更新することがあります。 重要な変更はこのページに反映し、必要に応じてリリースノートまたはアプリ UI でも案内します。
