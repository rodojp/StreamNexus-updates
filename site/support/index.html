Support

サポート

導入、更新、不具合報告の方法を案内します。

問い合わせ先

通常の質問や不具合報告は、 StreamNexus-updates の GitHub Issues で受け付けています。認証トークン、パスワード、個人情報、ログイン済み画面の画像は投稿しないでください。

  • 一般サポート: support@stream-nexus.com
  • 代表問い合わせ: info@stream-nexus.com
  • プライバシー / データ削除: privacy@stream-nexus.com
  • セキュリティ・脆弱性報告: security@stream-nexus.com
問い合わせを作成 問い合わせフォーム リリース一覧

動作要件

  • Windows 10 または Windows 11、64-bit
  • インターネット接続
  • Twitch・YouTube連携を利用する場合は、各サービスのアカウント
  • ベータ提供中は、Access Gateによる承認が必要になる場合があります。

インストールと更新

  1. 公式GitHub Releasesからインストーラーをダウンロードします。
  2. インストーラーを実行し、画面の案内に従います。
  3. 更新が公開された場合は、アプリ内の更新案内または公式GitHub Releasesを利用します。
  4. アンインストールはWindowsの「インストールされているアプリ」から行います。

不具合報告に含める情報

  • StreamNexusのバージョン
  • Windowsのバージョン
  • 発生手順と期待した動作
  • 再現頻度と、個人情報を除いたエラー内容

セキュリティに関わる内容は公開 Issue に投稿せず、 security@stream-nexus.com へ連絡してください。
