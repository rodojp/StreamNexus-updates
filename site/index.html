見たい配信を、ひとつの画面に。

StreamNexusは、TwitchとYouTubeのチャンネルを監視し、配信開始を通知します。 公開されている配信情報の確認と、複数配信のマルチビュー再生をひとつのアプリで行えます。

主な機能

ライブ配信監視

登録したTwitch・YouTubeチャンネルの配信状況を確認できます。

配信開始通知

登録チャンネルが配信を開始した際に、アプリから通知を受け取れます。

マルチビュー再生

複数の公開配信をひとつのデスクトップ画面で視聴できます。

YouTube API Servicesの利用方法

YouTube連携は、ユーザーがGoogle OAuthによる認可を明示的に完了した場合にのみ有効になります。 StreamNexusが要求するスコープは、読み取り専用の https://www.googleapis.com/auth/youtube.readonly のみです。

  • 認可したユーザーのYouTube登録チャンネルをStreamNexus内に表示します。
  • ユーザーが選択したYouTubeチャンネルのライブ配信状況を確認します。
  • 配信監視、通知、マルチビュー再生に必要な公開動画メタデータを表示します。

StreamNexusは、YouTube動画、コメント、再生リスト、チャンネルのアップロード、削除、編集、管理を行いません。 YouTube API Servicesのデータを販売、広告利用、無関係なプロファイリングに使用することもありません。
