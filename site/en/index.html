Core features

Track selected Twitch and YouTube channels and see their current live status.

Notifications

Receive app notifications when a tracked channel starts a live stream.

Multiview playback

Open multiple public streams in one desktop workspace.

How StreamNexus uses YouTube API Services

YouTube integration is enabled only after a user explicitly completes Google OAuth authorization. StreamNexus requests only the read-only scope https://www.googleapis.com/auth/youtube.readonly.

  • Display the authorized user's YouTube subscriptions inside StreamNexus.
  • Check live stream status for YouTube channels selected by the user.
  • Display public video metadata used for monitoring, notifications, and multiview playback.

StreamNexus does not upload, delete, edit, or manage YouTube videos, comments, playlists, or channels. It does not sell YouTube API Services data or use it for advertising or unrelated profiling.
