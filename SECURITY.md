# Security Policy

If you believe you have found a security vulnerability in StreamNexus or this public release site, email:

- <security@stream-nexus.com>

Do not open a public GitHub issue for security-sensitive reports.

Please include:

- Affected StreamNexus version or affected page.
- Operating system and app version, if applicable.
- Steps to reproduce.
- Expected and actual behavior.
- Potential impact.
- Relevant logs or screenshots with secrets, tokens, and personal data removed.

For general support, use <support@stream-nexus.com> or the public GitHub Issues page.
